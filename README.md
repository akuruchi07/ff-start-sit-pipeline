# Fantasy Football Start/Sit Pipeline

This project builds a data pipeline that helps answer a common fantasy football question:

**Which player should I start this week?**

The system ingests NFL player performance data and generates features to support weekly start/sit decisions.

## Project Goals

- Build a **data ingestion pipeline** using Python
- Store historical NFL player performance data
- Engineer predictive features (rolling averages, usage trends, matchup strength)
- Develop a model to compare two players and recommend who to start
- Eventually deploy the pipeline using **AWS serverless architecture**
